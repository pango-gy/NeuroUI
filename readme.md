## 🎯 Pango Neuro란?

Pango Neuro는 **광고 실무자**를 위해 설계된 AI 기반 분석 플랫폼입니다.

복잡한 명령어나 코딩 없이 **자연어**로 Google Ads 데이터를 분석하고, 인사이트를 얻을 수 있습니다.

### ✨ 주요 기능

- 🔗 **Google Ads 연동** - MCP(Model Context Protocol)를 통한 실시간 광고 데이터 조회
- 💬 **AI 어시스턴트** - 자연어로 데이터 분석 요청, 캠페인 성과 리포트 생성
- 📊 **데이터 시각화** - CSV, Excel 파일 분석 및 차트 생성
- 🔒 **로컬 데이터 보안** - 모든 대화 기록은 로컬 SQLite에 저장, 외부 서버로 전송되지 않음
- 🌐 **다중 AI 모델 지원** - Gemini, OpenAI, Claude 등 다양한 모델 사용 가능

## 🚀 시작하기

1. **설치** - 다운로드한 설치 파일 실행
2. **로그인** - Google 계정으로 로그인 또는 API Key 설정
3. **사용 시작** - 채팅창에 분석하고 싶은 내용 입력

### 예시 요청

```
"지난 30일간 캠페인별 CPC 추이를 보여줘"
"광고 성과가 가장 좋은 키워드 Top 10을 알려줘"
"이번 달 광고 예산 대비 실적 요약해줘"
```

## 🔧 설정

### Google Ads 연동

1. 설정 → 도구 → MCP 서버 추가
2. Google Ads MCP 서버 연결
3. OAuth 인증 완료

### AI 모델 설정

- **Gemini** (기본) - Google 계정으로 자동 연동
- **OpenAI** - API Key 입력
- **Claude** - API Key 입력
